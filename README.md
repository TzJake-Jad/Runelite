# Checklist Notes

A RuneLite plugin that provides a notepad with per-line checkboxes and paste support.

## Features

- Add notes one at a time or paste multiple lines at once
- Check off completed items with a strikethrough effect
- Delete individual notes with the ✕ button
- Clear all completed notes at once with the "Clear Done" button
- Notes are saved automatically and persist between sessions

## Usage

1. Open the **Checklist Notes** panel from the sidebar
2. Type or paste your notes into the text area at the bottom
3. Press **Add (Ctrl+Enter)** or click the button to add them
4. Check items off as you complete them
5. Use **Clear Done** to remove all checked items
